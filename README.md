# hashtable
Hash table project, written in c++
This program use Hash Table for managing the resoult of an examination, using Closed and Open Hash Table.
It is composed by 4 files:
1) main.cpp
2) common.h
3) tabellaaperta.h
4) tabellachiusa.h

It is localized in italian.
